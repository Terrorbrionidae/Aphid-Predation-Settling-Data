# Aphid Settling Behaviour Following Conspecific Predation
By: 22382300 <br />
Date of Experiment: 2023-12-12 to 2023-12-13 <br />
Date of Upload: 2024-01-21 <br />
Description: Data from experiment investigating Acyrthosiphon pisum settling behaviour following conspecific predation. <br />
Two leaf cuttings (from Vicia faba) placed into three petri dishes. <br />
First petri dish, leaves left as is. <br />
Second petri dish, one leaf coated with crushed aphid (simulate predation). <br />
Third petri dish, both leaves coated <br />
5-10 A. pisum introduced into each dish and settling behaviour observed after periods of time
